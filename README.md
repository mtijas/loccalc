# loccalc
Yet another Lines of Code calculator

## Running

Simply run against some directory or file (or many):
`$ python3 loccalc.py <directory/file> [<directory/file>...]`

## Testing 

Test by running `$ python3 loccalc.py manual-test`.

Should result:
```
** Totals **
LOC: 14
Comments: 55
Empty rows: 14
Total: 83
```
